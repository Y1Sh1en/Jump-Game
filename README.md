# Jump Jump Game 🎮

A fun charging jump game built with HTML5 Canvas. Press and hold to charge your jump power, release to jump! Test your precision and aim for the highest score!

## 🎯 Game Description

Control a cute little ball and jump onto red platforms. The longer you hold, the farther you jump. Land precisely on the platform to earn high scores and combo bonuses!

## 🕹️ How to Play

1. **Charge**: Hold down the left mouse button or SPACE key
2. **Jump**: Release the button to jump
3. **Scoring**:
   - Base score: 100-140 points
   - PERFECT hit: 1.5x bonus
   - Combo system: Consecutive successful jumps earn extra points
4. **Goal**: Achieve the highest score possible!

## ✨ Features

- 🎨 Beautiful visual effects and animations
- 📱 Touch screen support
- 🏆 Local high score saving
- ⚡ Real-time charging indicator and distance guide
- 🔥 Combo system with floating score animations
